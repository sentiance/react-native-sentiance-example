# RNSentiance Example

## Install npm dependencies
```sh
$ npm i
```


## iOS setup

### Install CocoaPods
```sh
$ gem install bundler
$ bundle install
```

### Instal iOS dependencies
```sh
$ cd ios && pod install
```

### Open project in XCode
```sh
$ open ios/RNSentiance.xcworkspace
```
